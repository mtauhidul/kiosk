# Patient Check-In Kiosk

Streamlined, self-service check-in for medical offices. Patients confirm info, snap insurance/ID, sign policies, and review everything—fast and friendly.

## What It Does

Guides patients through demographics, insurance, histories, e‑signature, and a quick survey. Built for kiosks and tablets with a clean, touch‑friendly UI.

## Highlights

- Smart camera capture for ID/insurance
- Complete intake: demographics, medical, meds, allergies
- Family, surgical, and social history
- E‑signature + final review
- Test mode for offline demo
- Responsive, fast, and simple

## Tech

React, MUI, CSS Modules, Redux, OpenCV.js, react‑webcam, Framer Motion
